# Plan 2.1: Scrapers (GitHub, Reddit, Blogs, Twitter Stub)

## Objective
Build the base scraper abstraction and concrete scrapers for GitHub repositories (via Search API), Reddit posts (via RSS), blog articles (via RSS), and a disabled Twitter stub. All scrapers output a unified item schema.

## Unified Item Schema
Every scraper must return a list of dicts with this shape:
```python
{
    "id": str,             # SHA256 hash of url
    "source": str,         # "github" | "reddit" | "blog" | "twitter"
    "title": str,
    "url": str,
    "description": str,    # Summary / body snippet
    "author": str,
    "metadata": dict,      # Source-specific (stars, forks, subreddit, etc.)
    "tags": list[str],     # Raw tags from source (topics, flair, etc.)
    "scraped_at": str,     # ISO 8601 timestamp
    "raw_content": str,    # Full text for LLM processing in Phase 3
}
```
